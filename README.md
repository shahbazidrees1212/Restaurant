# RestaurantMvcUltimatePro

Modern professional restaurant website using ASP.NET Core MVC, Razor Views, EF Core SQLite, Bootstrap 5, services and clean folders.

## Features
- Modern responsive homepage
- Full menu with search and category filter
- Table reservation form with validation
- Contact form stored in database
- Gallery page
- Events and catering page
- Chefs, offers and testimonials
- Admin dashboard preview for reservations/messages
- EF Core SQLite seed data

## Run in VS Code
```bash
dotnet restore
dotnet run
```
Open the localhost URL shown in terminal.

## Useful URLs
- `/Menu`
- `/Reservation/Create`
- `/Gallery`
- `/Events`
- `/Admin/Dashboard`
